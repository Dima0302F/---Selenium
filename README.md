# Login Tests Automation

Automated tests for the-internet.herokuapp.com login form using Selenium and pytest.

## Test Scenarios

1. **Successful Login** - Valid credentials (tomsmith / SuperSecretPassword!)
2. **Unsuccessful Login** - Invalid credentials (wrong username / wrong password)
3. **Empty Fields** - Login attempt with empty fields
